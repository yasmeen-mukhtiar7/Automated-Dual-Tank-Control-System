# Smart Dual-Tank Water Level Controller

## CodeAlpha Robotics & Automation Internship — Task 3

This repository contains the design and documentation of a Smart Dual-Tank Water Level Controller developed for the CodeAlpha Robotics & Automation Internship Program.

## Project Overview

The system is designed to automatically monitor and control water levels in two separate tanks using sensors and automatic motor control logic. The automation system helps prevent overflow and dry running while ensuring efficient water management.

## Objectives

- Design an automated water level control system
- Monitor upper and lower tank levels
- Automatically switch motor ON/OFF
- Reduce manual intervention
- Improve water usage efficiency

## System Components

- Water level sensors
- Control logic circuit
- Water pump motor
- Relay module
- Power supply
- Indicator system

## Working Principle

1. The lower tank water level is monitored continuously.
2. When the upper tank water level drops below the minimum threshold, the motor turns ON.
3. Water is pumped from the lower tank to the upper tank.
4. Once the upper tank reaches maximum level, the motor automatically turns OFF.
5. The system prevents overflow and protects the motor from dry running conditions.

## Features

- Automatic motor control
- Dual-tank monitoring
- Overflow prevention
- Dry-run protection
- Efficient water management
- Automation-based control system
